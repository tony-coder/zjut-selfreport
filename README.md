# zjut-selfreport
浙江工业大学每日一报挂机自动打卡



## 配置环境

1. 安装selenium+apscheduler

```shell
pip3 install selenium
pip3 install apscheduler
pip3 install pyvirtualdisplay
```

注意：服务器需安装chrome和对应驱动

## 运行

后台运行

```shell
nohup python3 daka.py &
```
